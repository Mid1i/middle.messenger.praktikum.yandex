# ME Chat

ME Chat — учебный проект веб-мессенджера.

Проект представляет собой SPA-приложение с собственной системой роутинга и набором переиспользуемых компонентов, реализованных на TypeScript и Handlebars. Интерфейс включает страницы авторизации, регистрации, профиля пользователя, списка чатов и окна переписки.

## Стек технологий

* TypeScript
* Handlebars
* SCSS
* Vite

## Функциональность

* Переиспользуемые UI-компоненты (кнопки, инпуты, ссылки, аватары и т.д.);
* Моковые данные для демонстрации интерфейса.

## Установка проекта

Клонировать репозиторий:

```bash
git clone https://github.com/Mid1i/middle.messenger.praktikum.yandex.git
cd middle.messenger.praktikum.yandex
```

Установить зависимости:

```bash
npm install
```

## Запуск проекта

Запуск локального сервера разработки:

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5173
```

## Сборка проекта

Сборка production-версии:

```bash
npm run build
```

Предпросмотр production-сборки:

```bash
npm run preview
```
