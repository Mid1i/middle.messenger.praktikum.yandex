# ME Chat

ME Chat — учебное клиентское SPA-приложение веб-мессенджера.

Интерфейс построен на TypeScript и Handlebars с использованием собственного базового
класса компонентов `BaseBlock`. Приложение содержит экраны авторизации, регистрации,
настроек профиля, списка чатов, переписки и ошибок. Данные пользователей, чатов и
сообщений на текущем этапе являются демонстрационными.

## Стек технологий

- TypeScript
- Handlebars
- SCSS
- Vite
- ESLint
- Stylelint
- Prettier

## Функциональность

- клиентская генерация страниц;
- переключение между экранами без перезагрузки страницы;
- страницы входа и регистрации;
- страница настроек пользователя и формы изменения данных и пароля;
- список чатов и лента сообщений активного чата;
- форма отправки сообщения;
- страницы ошибок 404 и 500;
- переиспользуемые компоненты кнопок, полей ввода, форм, ссылок, аватаров,
  всплывающих окон, элементов списка чатов и сообщений;
- единый механизм валидации форм по событиям `blur` и `submit`;
- проверка имени, логина, электронной почты, пароля, телефона и сообщения с помощью
  регулярных выражений;
- сбор и вывод данных отправленной формы в консоль;
- демонстрационные данные пользователей, чатов и сообщений.

## Архитектура

Проект разделён на слои MVC:

- `models` — демонстрационные данные пользователей, чатов и сообщений;
- `pages` и `components` — представления и переиспользуемые UI-компоненты;
- `controllers` — управление отображением и переключением страниц;
- `services` — переиспользуемая бизнес-логика, включая валидацию.

`BaseBlock` отвечает за рендеринг Handlebars-шаблонов, работу с дочерними
компонентами, событиями, жизненным циклом и обновлением при изменении `props`.

## Установка проекта

Клонировать репозиторий:

```bash
git clone https://github.com/Mid1i/middle.messenger.praktikum.yandex.git
cd middle.messenger.praktikum.yandex
```

Установить зависимости:

```bash
npm install
```

## Запуск проекта

Запуск локального сервера разработки:

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5173
```

## Сборка и production-запуск

Сборка и запуск production-версии:

```bash
npm run start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Проверка качества кода

Запуск проверки типов, ESLint, Stylelint и Prettier:

```bash
npm run lint
```

Команда последовательно запускает проверку типов TypeScript, ESLint, Stylelint и
проверку форматирования Prettier.

Отдельная проверка типов:

```bash
npm run typecheck
```

Автоматическое форматирование поддерживаемых файлов:

```bash
npm run format
```
